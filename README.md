# Pwn-Box

A docker environment for Pwning in CTF based on **phusion/baseimage**

## Usage

Use interactive script:

```
bash ./start.sh
```

## Include tools

- Capstone
- CMake
- GNU Debugger
- IPython
- ltrace & strace
- NASM
- NetCat
- net-tools
- OneGadget
- pwndbg
- pwngdb
- pwntools
- Python2 & 3
- ROPgadget
- Ruby 2.3
- tmux
- Wget

持續更新 (如果有上手新的工具)
